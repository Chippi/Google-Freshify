Quick commands

You can modify the date filter by using quick text commands. For example, you can type funny cats 1y to set the filter to 1 year.

Available commands:

  • {number}h - filter by hour
  • {number}d - filter by day
  • {number}m - filter by month
  • {number}y - filter by year

Example: funny cats 1d

Clear filter

You can decide if you wish to clear the filter options after x amount of minutes. To always remember, set the field to 0 or empty

Filter will be cleared after minutes.
